# FORMA — AI фитнес-платформа (интерактивный прототип)

Маркетплейс фитнес-тренеров с **AI-разбором техники**. Кликабельный мобильный веб-прототип для проверки гипотезы — **два приложения в одном**: для клиента и для тренера. Все данные замоканы.

> Рынок по ТЗ — Узбекистан / Средняя Азия (платежи Payme / Click / Uzcard / Humo, цены в сумах, OTP по номеру).

## Что внутри

**Приложение клиента**
- Маркетплейс тренеров: поиск, фильтры, профили, портфолио, отзывы
- Подписка и оплата (Payme / Click / Uzcard / Humo) — с экраном успеха
- Программа тренировок и тренировочный день
- 🎯 **AI Form Coach** — разбор техники: скелет поверх «камеры», подсчёт повторений, оценка формы, голосовые подсказки → итог тренировки (FormSession)
- AI-ассистент (вопросы про боль/технику, видео-демо, дисклеймеры)
- Чат с тренером (текст / видео / голос)
- Прогресс: сила, вес, оценка техники, объём, замеры
- Питание: меню, калории и БЖУ
- Профиль, уведомления

**Приложение тренера**
- Дашборд: доход, клиенты, расписание, запросы
- База клиентов и карточка клиента (прогресс, разборы)
- Контент: программы и меню, лёгкий редактор программы
- Финансы: баланс, выплаты, операции, комиссия платформы
- Входящие и чат с клиентом
- Профиль/портфолио (редактор + предпросмотр)

## Стек

React + Vite + TypeScript · Tailwind CSS · Framer Motion · React Router (HashRouter) · lucide-react.
Фото — Unsplash. Анимации — Framer Motion. AI-разбор техники — кастомный SVG-скелет.

## Запуск локально

```bash
npm install
npm run dev
```

На десктопе — телефон по центру; на телефоне — на весь экран.

## Сборка

```bash
npm run build      # -> dist/
npm run preview
```

## Деплой

Пуш в `main` автоматически публикует на GitHub Pages (см. `.github/workflows/deploy.yml`).
`base: './'` в `vite.config.ts` + HashRouter — работает на любом подпути.
